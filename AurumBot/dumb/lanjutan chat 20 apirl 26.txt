### LANGKAH 3: Parameter yang Disepakati

- Timeframe entry: M1
- Timeframe bias: M15

- Min Distance: 20 points
- Magic Number: 123456
- Max Floating Loss: -50 USD
- RR Ratio: 1:3 (ATR-based)
- Fase ujicoba threshold:
  * Fase 1: 0.65 | maks 4 hari / min 50 trade
  * Fase 2: 0.72 | maks 4 hari / min 50 trade
  * Fase 3: 0.82 | maks 4 hari / min 50 trade

### LANGKAH 4: Workflow Bot (6 Lapis Filter — setelah simplifikasi)
Layer 1: M15 Strong Trend Lock (pause kalau slope > 15)
Layer 2: M15 Bias Filter (bullish/bearish/sideways)
Layer 3: Key Level Check (near_support / near_resistance, radius 8 pts)
Layer 4: S&D Zone OR Candlestick Pattern (in_sd_zone or candle_confirm)
Layer 5: Rule-Based Signal (MACD crossover + RSI kondisi)
Layer 6: ML Confirmation (prob >= threshold 0.65)
+ Position Management (MAX_TRADE, MIN_DISTANCE)

Filter yang DINONAKTIFKAN untuk fase demo:
- BoS (Break of Structure) filter → terlalu ketat, 0 trade
- Volume filter → vol_ok selalu False, 0 trade

### LANGKAH 8: Training Model (Banyak Iterasi)

Iterasi 1: Fixed TP=21/SL=14, filter key level radius=5
→ BUY label=1: 661 (1.3%), SELL=904 (1.8%)
→ PF: 0.21 → GAGAL

Iterasi 2: Naikkan KEY_LEVEL_RADIUS=15
→ BUY=809, SELL=1142
→ PF: 0.21 → masih GAGAL

Iterasi 3: Ganti RR ke 1:3 (TP=39/SL=13)
→ Natural win rate: 0.1% (sangat rendah)
→ PF: 0.33 → masih GAGAL

Iterasi 4: Hapus semua filter dari labeling (pure mode)
→ Natural win rate tetap 0.1% → TP terlalu besar vs ATR data

Iterasi 5: ATR-adaptive labeling (TP=3xATR, SL=1xATR)
→ Natural win rate: 25.8% BUY, 24.9% SELL
→ Simulasi hanya 1 trade BUY → Mean ATR=0.87 terlalu kecil

Iterasi 6: Naikkan ATR multiplier (TP=15xATR, SL=5xATR untuk labeling)
→ Natural win rate: 2.9% BUY, 3.1% SELL → terlalu kecil lagi

Iterasi 7: ATR_MULTIPLIER_TP=4.0, ATR_MULTIPLIER_SL=1.0 (runtime bot)
dengan labeling TP=15x/SL=5x
→ PF: 1.01 ✅ BREAK EVEN
→ Best combination: TP=4xATR / SL=1xATR

### LANGKAH 9: Menjalankan Bot

Percobaan 1: Hari Sabtu → market tutup, harga stuck
→ Baru sadar forex tutup Sabtu-Minggu

Percobaan 2: Senin pagi → bot jalan
→ Masalah: ATR calculation failed (warning sering muncul)
→ Solusi: naikkan candle dari 100 ke 200

Percobaan 3: BoS filter aktif → 0 trade dalam 2+ jam
→ prob_buy sempat 0.88-0.92 tapi bos_bull selalu False
→ Solusi: NONAKTIFKAN BoS filter dari kondisi entry

Percobaan 4: Volume filter aktif → vol_ok selalu False
→ 0 trade meskipun BoS sudah dinonaktifkan
→ Solusi: NONAKTIFKAN volume filter

Percobaan 5 (SEKARANG): Bot running ~21:45 WIB
→ Filter aktif: M15 bias + Key Level + S/D + MACD/RSI + ML
→ Masih menunggu trade pertama
